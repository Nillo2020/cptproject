What is Software as a Service?

  1. cloud providers deliver a computing platform typically including operating system, programming language execution environment, database, and web server.
  2. cloud providers install and operate application software in the cloud and cloud users access the software from cloud computing.
  3. cloud providers offer computers, as physical or more often as virtual machines, and other resources.
  4. the practice of using a network of remote servers hosted on the Internet to store, manage, and process data, rather than a local server or a personal computer.
